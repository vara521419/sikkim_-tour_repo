# Sikkim Smart Tourism Website 🏔️

A modern, interactive tourism website for Sikkim featuring AI-powered chatbot, beautiful animations, and comprehensive user experience enhancements.

## 🛠️ Setup Instructions

1. **Clone or Download** the project files
2. **API Configuration**: Your Gemini API key is already configured
3. **Local Server**: Run with Python HTTP server or any web server
4. **Browser**: Open in modern browser (Chrome, Firefox, Safari, Edge)

### Running Locally
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have live-server)
npx live-server

# Using PHP
php -S localhost:8000
```
